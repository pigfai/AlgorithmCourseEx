### 1.1 排序
#### 问题描述：

对随机生成的若干组整数序列（比如规模为1000个数，10000个数，100000个数）进行排序，排序算法使用四种方法，插入排序算法、合并排序算法和快速排序算法

#### 重点问题和解决方法：

1. 随机生成数字

使用rand() 生成，但是有局限在于 只能生成 0 - 32767 大小的范围，所以又进行了减法和缩放，获得了正负都有的随机数

long num = rand() % 32767 - 16383;

int mul = rand() % 100 + 1;

a[i] = num * mul;

2. 统计算法时间

使用  GetTickCount64();  毫秒级
